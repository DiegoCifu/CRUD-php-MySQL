# CRUD PHP & MySQL

Aplicación CRUD desarrollada con PHP y MySQL como parte de mi portafolio profesional.

## Funcionalidades
- Login de administrador con sesiones
- Crear, editar y eliminar proyectos
- Subida de imágenes
- Protección de rutas
- Hash de contraseñas
- Seguridad contra SQL Injection

## Tecnologías
- PHP
- MySQL
- Bootstrap 5
- HTML5
- CSS3

## Instalación
1. Clonar el repositorio
2. Importar `database.sql` en phpMyAdmin
3. Configurar `conexion.php`
4. Ejecutar el proyecto en XAMPP
